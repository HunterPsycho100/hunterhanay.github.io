Course Flowchart

This local project was built using the React library. It is a flowchart that shows the most efficient class schedule for your major.

The application takes in a .txt file with all the major's course requirements (class code, name, prereqs, quarter availability, and credits). I parsed the info into JSON format which made the data clean and easier to access for future functions.

I then created an adjacency list by first creating a list of all the class codes. Those class codes would be used to access the JSON information and whenever a class could have any prereqs, I would add the current class to the adjacency list that was associated to the prereqs.

I then took that adjacency list and put it into a topological sort which puts it in a list where all the nodes are pointed in one direction.

I then populated a queue in in that topological order with the class codes. I looped through the queue and validated if the class item at the front of the queue passed the prereq requirements, the credit space for that quarter, and whether the class was available for that quarter. If the class item did pass, then it would be added to the class schedule and marked as visited. If the class did not pass the validation process then it was appended to the queue.

After opening the application in your localhost server, you are presented with an input of the major's txt file, how many credits you would like to take per quarter and what quarter you would like to begin in. Then you hit submit, and a flowchart would be populated with the courses organized by year and quarter.
